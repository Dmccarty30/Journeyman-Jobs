Circuit Background System

üîå ElectricalCircuitBackground Component

A performance-optimized animated circuit board background that provides the signature electrical theme across all screens.

Density Levels:

  • ComponentDensity.low - Minimal circuit traces (0.5x multiplier)
  • ComponentDensity.medium - Standard screen background (1.0x multiplier)
  • ComponentDensity.high - Dense circuit pattern (1.5x multiplier)
  • ComponentDensity.ultra - Maximum density (2.0x multiplier)
Standard Configuration: 
ElectricalCircuitBackground(
  opacity: 0.08,
  componentDensity: ComponentDensity.medium,
  enableCurrentFlow: true,
)

Best Practices & Guidelines

Design System Rules

üéØ Golden Rules for UI Consistency
  1. Always use AppTheme constants - Never hardcode colors, spacing, or design values
  2. Reference existing patterns - Check Job Card or Local Card for implementation examples
  3. Medium density backgrounds - Use ComponentDensity.medium for all screen backgrounds
  4. Copper borders at 1.5px - Standard card border is borderWidthMedium
  5. Standard card shadow - Use AppTheme.shadowCard for all card components
  6. 12px border radius - Use AppTheme.radiusMd for cards
