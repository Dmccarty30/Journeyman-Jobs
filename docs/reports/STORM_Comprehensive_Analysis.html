üí° Lessons Learned & Best Practices

Technical Insights

1. Continuous Animation Testing Challenge

Problem: Circuit background animation caused pumpAndSettle() timeouts in performance tests.

Solution: Used fixed-duration pump() calls instead of pumpAndSettle() to measure stable state after initial frame.

Lesson: Animated UIs require specialized testing approaches that account for continuous state changes.

2. Exception Handler File Path Bug

Problem: Checking line content for "class AppTheme" caused 202 false positives in definition files.

Solution: Changed function signature to accept file path and check path instead of line content.

Lesson: Context-aware exception handling requires file path information, not just line content analysis.

3. Accessibility Color Contrast

Problem: ADMIN badge with warningYellow background had 3.4:1 contrast ratio (WCAG failure).

Solution: Changed to errorRed background achieving 4.13:1 contrast ratio (WCAG AA compliant).

Lesson: Color choices must balance visual design intent with accessibility requirements. Test early.
