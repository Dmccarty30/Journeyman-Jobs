Journeyman Jobs is a specialized job discovery app designed exclusively for IBEW (International Brotherhood of Electrical Workers) union members. The app connects journeymen, linemen, wiremen, and other electrical professionals with contractors across North America.

Flutter 3.x Firebase Suite Riverpod State Management Go Router Navigation NOAA Weather API Google/Apple Sign-In

Core Mission

"Clearing the Books" - The app's tagline reflects its purpose of helping electrical workers find job opportunities quickly and efficiently, reducing time spent on unemployment lists ("the books").

üöÄ Key Technical Decisions

Framework Choices

  • Flutter: Cross-platform with excellent performance for complex animations
  • Firebase: Comprehensive backend-as-a-service with offline support
  • Riverpod: Robust state management with dependency injection
  • GoRouter: Type-safe navigation with nested routing support
  • NOAA APIs: Official government weather data (no API keys required)

Jobs and locals load incrementally with scroll pagination
