Tailboard Tabbed Interface Design Specification

Comprehensive design system for the new tabbed interface in Journeyman Jobs

Design System Reference
This specification is based on the existing AppTheme design system located in lib/design_system/app_theme.dart and follows all established electrical-themed design patterns.

1. Architecture Overview

1.1 Component Hierarchy

The tabbed interface consists of four primary layers:

  1. Header Section - Group information and settings (existing)
  2. Dynamic Container Row - 4 context-sensitive containers
  3. Tab Bar - Main navigation (Feed | Jobs | Chat | Members)
  4. Content Area - Tab-specific content
  5. Bottom Navigation - App-wide navigation (existing)

5.2 Text Overflow Handling

  • 1-word labels: Full display, centered
  • 2-word labels: Two lines if needed, centered
  • Long labels: Ellipsis after 2 lines (overflow: TextOverflow.ellipsis, maxLines: 2)
  • Minimum readable: 70px container width enforced

10. Performance Considerations

10.1 Optimization Strategies

  • Container Updates: Use const widgets where possible
  • Tab Switching: Instant container swap (no animation overhead)
  • Gradient Rendering: Cache gradient objects in build method
  • Text Measurement: Pre-calculate text sizes for common labels
  • State Management: Use local state for tab index (no Provider needed)

10.2 Build Performance

  • Container row rebuilds only on tab change (4 containers updated)
  • Tab bar rebuilds only on selection change
  • Content area uses IndexedStack to preserve state
  • Avoid rebuilding entire screen on tab switch

11.1 Widget Tests

13. Code Snippets

13.1 Container Builder

Widget _buildContainer(String label) {
  return Expanded(
    child: Container(
      height: 56,
      decoration: BoxDecoration(
        gradient: AppTheme.buttonGradient,
        borderRadius: BorderRadius.circular(AppTheme.radiusMd),
        border: Border.all(
          color: AppTheme.borderCopper,
          width: AppTheme.borderWidthCopper,
        ),
        boxShadow: AppTheme.shadowCard,
      ),
      child: Material(
        color: Colors.transparent,
        child: InkWell(
          onTap: () => _handleContainerTap(label),
          borderRadius: BorderRadius.circular(AppTheme.radiusMd),
          child: Center(
            child: Padding(
              padding: EdgeInsets.symmetric(
                horizontal: 8,
                vertical: 12,
              ),
              child: Text(
                label,
                style: AppTheme.labelMedium.copyWith(
                  color: AppTheme.white,
                ),
                textAlign: TextAlign.center,
                overflow: TextOverflow.ellipsis,
                maxLines: 2,
              ),
            ),
          ),
        ),
      ),
    ),
  );
}

13.2 Dynamic Container Row

Widget _buildDynamicContainers() {
  List<String> currentContainers = _containerData[_selectedTabIndex] ?? [];

  return Padding(
    padding: EdgeInsets.symmetric(
      horizontal: AppTheme.spacingMd,
      vertical: AppTheme.spacingSm,
    ),
    child: Row(
      children: [
        for (int i = 0; i < currentContainers.length; i++) ...[
          _buildContainer(currentContainers[i]),
          if (i < currentContainers.length - 1)
            SizedBox(width: AppTheme.spacingSm),
        ],
      ],
    ),
  );
}
