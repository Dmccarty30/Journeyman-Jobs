Different roadblocks require different techniques. Use this skill to identify the correct approach when you are stuck.

## The Process
1. **Identify stuck-type**: Match your symptoms to the table above.
2. **Load the skill**: Read the specific instructions for that technique.
3. **Apply technique**: Follow the process exactly.
4. **Combine if needed**: Use multiple techniques (e.g., find a pattern then simplify) if the first doesn't solve it.

**Don't force through a roadblock with the same failing approach. Stop, identify the type of "stuck", and switch techniques.**
